#  Smart Wardrobe & Digital Inventory Management (Akıllı Gardrop ve Envanter Yönetimi)

Bu proje, kullanıcıların kişisel envanterlerini dijital ortamda organize etmelerini, takip etmelerini ve yönetmelerini sağlayan kapsamlı bir yazılım mimarisi projesidir. İstemci (mobil) ve sunucu (backend) tabanlı çalışan sistem; veri yönetimi, kullanıcı dostu arayüz tasarımı ve dijital varlık takibine odaklanmaktadır.

##  Proje Amacı ve Mimari Yaklaşım
* **Dijital Envanter Yönetimi:** Fiziksel varlıkların (kıyafetlerin) dijital bir veri tabanında kategorize edilerek saklanması ve algoritmik olarak eşleştirilmesi.
* **Modern Sistem Tasarımı:** Mobil uygulama ile asenkron çalışan hızlı bir API sunucusunun (Client-Server mimarisi) kurgulanması.
* **Akıllı Analiz:** Yapay zeka entegrasyonu sayesinde envanter verilerinin analiz edilerek kullanıcıya akıllı kombin önerileri ve durum takibi sunulması.

##  Kullanılan Teknolojiler
* **Frontend (Mobil):** Flutter / Dart (Cross-platform mobil uygulama geliştirme)
* **Backend (Sunucu):** Python / FastAPI (Yüksek performanslı, asenkron RESTful API yönetimi)
* **Yapay Zeka:** Gemini API Entegrasyonu (Akıllı eşleştirme ve veri işleme)
* **Sistem Tasarımı:** Veri Modelleme, Temiz Kod (Clean Architecture) ve API Tabanlı Envanter Takip Algoritmaları

##  Temel Özellikler
* Varlıkların sisteme kaydedilmesi, veri tabanında güncellenmesi ve kategorik bazda listelenmesi.
* Hızlı yanıt veren (low-latency) FastAPI arka planı ile kesintisiz veri akışı.
* İleride eklenebilecek yeni modüllere (örn. e-ticaret entegrasyonu veya IoT cihazları) uygun, genişletilebilir proje altyapısı.
